# ReproRun

> Дайте ему статью. Он скажет вам, воспроизводятся ли результаты на самом деле.

**ReproRun** — это переносимый **навык AI-агента**, который может использовать любой совместимый AI-ассистент для программирования. Он автоматизирует мучительный путь от *«статья утверждает X»* до *«X действительно запускается на моей машине»*. Статьи публикуют красивые цифры; их воспроизведение обычно умирает на сломанном коде, несобираемых окружениях и расхождении версий. ReproRun берёт на себя весь конвейер от начала до конца:

**прочитать статью → найти код и данные → собрать окружение → smoke-тест → полный запуск → сравнить измеренные цифры с утверждениями статьи.**

---

## ✨ Создан, чтобы адаптироваться — на любой платформе

ReproRun спроектирован так, чтобы **самостоятельно подстраиваться под то, что ему передают**, а не предполагать одну фиксированную конфигурацию:

- **Кроссплатформенность по ОС** — работает на Windows, macOS и Linux
- **Поддержка разных языков** — полный конвейер для Python; минимальный путь для R / MATLAB / Julia
- **Поддержка разных областей** — биология одиночных клеток, машинное обучение для изображений и другое, без перенастройки под каждую область
- **Поддержка разных режимов** — *режим инструмента* (`pip install` + написание вызывающего скрипта) или *режим эксперимента* (клонирование репозитория и запуск его скриптов) — выбирается автоматически для каждой статьи

---

## 🚀 Что он делает

- **Находит статью только по её названию** — автоматический поиск в интернете PDF и репозитория кода
- **Автоматически диагностирует деградацию окружения** — конфликты ABI numpy, изменения API torchvision, устаревшие методы pandas… обнаруживаются и исправляются автоматически
- **Никаких догадок по параметрам** — проверяет сигнатуры функций после установки
- **Цикл самоисцеления зависимостей из 5 раундов** — классификация ошибки → целевое исправление → повторная проверка, до 5 раундов
- **Изоляция статей** — каждая статья получает собственное пространство имён для вывода

## 🏗️ Архитектура

Один оркестратор (`SKILL.md`) управляет 6 специализированными агентами:

| Агент | Роль |
|-------|------|
| A · paper-reader | Извлекает числовые утверждения для воспроизведения |
| B · resource-finder | Находит репозиторий кода и наборы данных |
| C · environment-builder | Собирает и чинит среду выполнения (самый сложный) |
| D · smoke-tester | Быстрый smoke-тест — подтверждает, что всё запускается |
| E · full-runner | Полный запуск воспроизведения |
| F · result-comparator | Сравнивает измеренное с заявленным, пункт за пунктом |

## 📦 Начало работы

ReproRun — это навык агента, который работает с любым совместимым AI-ассистентом для программирования. Чтобы его использовать:

1. Используйте AI-агента для программирования, способного загружать навыки.
2. Поместите папку `paper-reproduction/` туда, где ваш агент загружает навыки.
3. В сессии просто попросите — например, *«воспроизведи scVelo»* или *«воспроизведи Таблицу 2 из этой статьи»*. Навык запускается автоматически.
